## 首次 Release 内容

SY Codex 助手是一款面向中国用户的 Codex 入门安装与配置工具，帮助新手用户完成 Codex 桌面 App 安装、代理 API 配置、中文增强和版本更新。

### 功能亮点

- Codex 安装向导。
- API Base URL、Key、协议和模型配置。
- 写入 Codex 本地 `config.toml` 和 `auth.json`。
- Codex 菜单和页面运行时中文增强。
- 软件内在线更新：下载进度、缓存复用和 SHA256 校验。
- 意见反馈页面。
- OpenClaw 入口预留。
